# Archive de nettoyage — 2026-07-15

Cette archive est préparée pour les scripts manuels et prototypes dont le
remplacement est confirmé. Aucun fichier actif n’a été déplacé pendant cette
passe : les `*_test.go` restent à leur package, et les outils encore utiles
restent dans `tools/` jusqu’à une migration dédiée.

Avant toute suppression future, ajouter ici le fichier concerné et préciser :

- son origine et son usage historique ;
- le test, diagnostic ou module qui le remplace ;
- la preuve qu’aucune documentation ou cible Make ne le référence ;
- la date à laquelle sa suppression pourra être réévaluée.
