# Tests, fixtures et scénarios

Les tests Go et frontend restent près du code qu’ils vérifient. Ce dossier
accueille uniquement les ressources partagées :

- `fixtures/` : payloads déterministes sans secrets ;
- `scenarios/` : scénarios d’intégration réutilisables ;
- `integration/` : ressources de tests d’intégration non liées à un package ;
- `system/` : sorties et helpers de tests système versionnables ;
- `archived/` : anciens artefacts conservés avec README et date.

Synora Lab est un module produit/admin et ne doit pas être déplacé ici.
